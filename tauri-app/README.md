# YLS Desktop Workbench

This directory contains the active Tauri desktop application for YLS usage analysis.

## Stack

- Tauri v2
- Vue 3
- TypeScript
- Bun
- Rust

## Current Scope

- Desktop shell and build wiring
- Rust backend entrypoint
- Vitest setup for the Vue shell

## Commands

- `bun install`
- `bun run test`
- `cargo check --manifest-path src-tauri/Cargo.toml`
- `bun run tauri dev`
